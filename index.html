PROJECTS

These are a few projects I've worked on recently. For older projects, please refer to the full project list.

Glowing Bear
Glowing Bear is a modern IRC client, or, more specifically, a browser/app frontend. It is implemented as a WeeChat relay client using WebSockets, and works great both in desktop and mobile environments.

instavpn
Ever needed a VPN right now? Using DigitalOcean and sshuttle, instavpn provides you with a dirt cheap working VPN within sixty seconds. It'll even tear down the host once you're done, so you only pay for what you use.

hubot-tell
Hubot-tell is a hubot plugin for relaying messages to users that are currently not present.
